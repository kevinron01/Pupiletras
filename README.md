# Pupiletras

Juego de sopa de letras para Android con 3 niveles de dificultad.

## Niveles

1. Fácil — Animales (cuadrícula 9×9)
2. Medio — Frutas (cuadrícula 11×11)
3. Difícil — Países (cuadrícula 13×13)

## Compilar

Abre el proyecto en Android Studio, o desde terminal:

```
gradle wrapper   # primera vez, para generar gradlew
./gradlew assembleDebug
```

Requiere Android SDK con `compileSdk` 34.
